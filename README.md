Objective
The goal of this task is to understand how to:
- Load a dataset using pandas
- Explore its structure and summary statistics
- Visualize data using different plots

Approach

1. Data Loading
   - Loaded the Iris dataset from a CSV file using pandas.

2. Data Exploration
   - Used `.shape` to understand dataset size
   - Used `.columns` to view feature names
   - Used `.head()` to preview data
   - Used `.describe()` for statistical summary

3. Data Visualization
   - Scatter Plot: Analyzed relationships between sepal and petal lengths
   - Histogram: Examined distribution of each feature
   - Box Plot: Identified outliers and spread of data

Results and Insights

- The dataset contains multiple features related to flower measurements.
- Scatter plots show clear separation between species based on petal length and width.
- Histograms reveal that most features follow a roughly normal distribution.
- Box plots highlight potential outliers and variation across features.

Conclusion
This task demonstrates how simple visualizations can provide meaningful insights into a dataset. The Iris dataset is well-structured and ideal for practicing exploratory data analysis (EDA).
